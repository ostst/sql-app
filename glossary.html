Основные понятия AI и нейросетей

A C D F G L M N P R T
A

AI (Artificial Intelligence)

Искусственный интеллект — область компьютерных наук, которая занимается созданием систем, способных выполнять задачи, требующие человеческого интеллекта: распознавание речи, принятие решений, перевод языков.

API (Application Programming Interface)

Программный интерфейс — набор правил и протоколов, позволяющих различным программам взаимодействовать друг с другом. Через API можно интегрировать нейросети в свои приложения.

C

Chatbot (Чат-бот)

Программа, которая имитирует разговор с пользователем на естественном языке. Современные чат-боты на базе LLM способны вести сложные диалоги и выполнять разнообразные задачи.

Context Window (Контекстное окно)

Максимальное количество токенов, которое модель может обработать за один раз. Определяет, сколько текста модель «помнит» в рамках одного диалога.

D

Deep Learning (Глубокое обучение)

Метод машинного обучения, использующий многослойные нейронные сети. Позволяет моделям самостоятельно выявлять сложные паттерны в данных.

Diffusion Model (Диффузионная модель)

Тип генеративной модели, которая создаёт изображения путём постепенного «очищения» шума. Используется в Midjourney, DALL-E, Stable Diffusion.

F

Fine-tuning (Дообучение)

Процесс дополнительного обучения предобученной модели на специфических данных для улучшения её работы в конкретной области.

G

GPT (Generative Pre-trained Transformer)

Семейство языковых моделей от OpenAI. GPT-4 — одна из самых мощных LLM, способная генерировать текст, код, анализировать изображения.

Generative AI (Генеративный ИИ)

ИИ-системы, способные создавать новый контент: текст, изображения, музыку, видео. Включает LLM, диффузионные модели и другие генеративные архитектуры.

L

LLM (Large Language Model)

Большая языковая модель — нейросеть с миллиардами параметров, обученная на огромных объёмах текста. Примеры: GPT-4, Claude, GigaChat, LLaMA.

M

Machine Learning (Машинное обучение)

Область ИИ, где системы учатся на данных без явного программирования. Модели выявляют паттерны и делают прогнозы на основе обучающих примеров.

Multimodal (Мультимодальный)

Модели, способные работать с несколькими типами данных одновременно: текст, изображения, аудио. Пример: GPT-4V может анализировать изображения и текст.

N

Neural Network (Нейронная сеть)

Вычислительная система, вдохновлённая биологическим мозгом. Состоит из слоёв «нейронов», которые обрабатывают информацию и учатся на примерах.

NLP (Natural Language Processing)

Обработка естественного языка — область ИИ, занимающаяся взаимодействием компьютеров с человеческим языком: понимание, генерация, перевод текста.

P

Prompt (Промпт)

Текстовая инструкция или запрос, который пользователь отправляет нейросети для получения нужного результата. Качество промпта напрямую влияет на качество ответа.

Prompt Engineering (Промпт-инженерия)

Искусство и наука составления эффективных промптов. Включает техники: few-shot learning, chain-of-thought, role prompting и другие методы улучшения результатов.

R

RAG (Retrieval-Augmented Generation)

Метод, при котором LLM дополняется внешней базой знаний. Модель сначала ищет релевантную информацию, затем генерирует ответ на её основе.

T

Token (Токен)

Базовая единица текста для LLM. Один токен — примерно 4 символа или 3/4 слова на английском. На русском — обычно меньше из-за кириллицы.

Transformer (Трансформер)

Архитектура нейросети, использующая механизм внимания (attention). Основа всех современных LLM — GPT, BERT, Claude и других.

Temperature (Температура)

Параметр, управляющий «креативностью» модели. Низкая температура (0.1-0.3) — предсказуемые ответы, высокая (0.7-1.0) — более разнообразные и творческие.
